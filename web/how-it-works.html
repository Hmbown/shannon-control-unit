The MDL Principle

Minimum Description Length (MDL) says the best model is the one that provides the shortest description of the data.

Think of it like file compression:

  • A random file can't be compressed (no patterns)
  • A file of all zeros compresses to tiny size (simple pattern)
  • Good models find compressible patterns in data

Shannon implements this by making the model "pay" in bits:

  • Bits to store the model weights (parameter cost)
  • Bits to store the prediction errors (data cost)
  • Total = Parameter Bits + Data Bits ‚Üê Minimize this!
