Simple, Practical NCD

NCD: compression‑based similarity (approximates shared structure; not semantics)

def understand(A, B):
    size_a = compress(A)
    size_b = compress(B)
    size_ab = compress(A + B)
    
    distance = (size_ab - min(size_a, size_b)) / max(size_a, size_b)
    return distance
