Evaluation Methodology

Rigorous, reproducible benchmarking

Experimental Setup

  • Hardware: Apple M4 Max, 36GB RAM, macOS (Darwin 25.0.0)
  • Baselines: IsolationForest, One-Class SVM, LOF, Cognition (all defaults)
  • Metrics: Precision, Recall, F1-Score, AUROC, AUPRC, Latency
  • Datasets: 19 total (13 security, 4 biomedical, 2 industrial)
  • Reproducibility: Full code and configs at github.com/hmbown/shannon-control-unit

Reproduce Results

# Clone repository
git clone https://github.com/hmbown/shannon-control-unit.git
cd shannon

# Install dependencies
pip install -r requirements.txt

# Run full evaluation
python datasets/evaluate.py --auto-tune --save-best

# Results saved to:
# - datasets/eval_results.json
# - datasets/config/params.json
